# 工作方式
1.  **搜索**：使用 `web_search_google` 搜索 `site:wiki.biligame.com/umamusume` 和 `site:mzh.moegirl.org.cn`。
2.  **爬取**：优先爬取 Bilibili Wiki（数据全），辅以 萌娘百科（梗和杂谈多）。使用视觉提取或文本提取工具获取全部内容。
3.  **数据处理原则（至关重要）**：
    * **禁止简写**：不要把“在菊花赏中以微弱优势战胜了对手”这种概括性的话作为唯一描述，要保留“在最后直线与xx进行了激烈的争夺，最终以颈差获胜，打破了xx的记录”这样的细节。
    * **保留原文**：对于简介、性格描述、名台词，**直接摘录原文**，不要用自己的话重写。
    * **结构化**：对于数值、适应性、技能等，必须使用 Markdown 表格完整保留。

请从网页中提取以下所有维度的信息：

    * **技能列表**（固有技能、觉醒技能的名称）。
2.  **角色塑造（深度挖掘）**：

    * 性格侧写（不仅是关键词，要包含具体的行为举止、心理活动描述）。
    * **语音与台词**：提取具有代表性的 3-5 句台词（如登录语音、比赛胜利语音），展现其语气口癖。
3.  **生涯经历（详细叙述）**：
    * 按时间顺序整理其赛马娘剧情（出道 -> 经典级 -> 资深级）。
    * **关键比赛**：不要只列结果。描述比赛的背景（如“无声铃鹿的复出战”）、对手、比赛过程中的关键转折。
    * **特殊事件**：如温泉事件、专属称号获取条件及含义。
4.  **人际关系（具体互动）**：
    * 不仅列出名字，要详细描述“她们之间发生过什么故事”、“互相对对方的评价”。
5.  **原型考据（详实）**：
    * 史实马的完整生涯（出生、主要胜鞍、退役原因、种马生涯、去世时间）。
    * **史实与动画/游戏的联系**：比如“游戏中的某个技能致敬了史实中的哪场比赛”。
